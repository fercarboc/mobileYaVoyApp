# Implementación HELPER - Plan de Trabajo

## 1. HomeScreen - Ver Anuncios ✅ (Actualizar)
- Cargar jobs desde VoyJobs con status='OPEN'
- Filtros por categoría
- Búsqueda por texto
- Ver si ya aplicó (VoyJobApplications)
- Botón para ver detalles y aplicar

## 2. JobDetailScreen - Detalle + Aplicar
- Mostrar detalles completos del job
- Formulario de candidatura (mensaje, precio propuesto)
- Enviar a VoyJobApplications
- Estado: PENDING

## 3. MyJobsScreen - Mis Candidaturas
- Lista de trabajos a los que aplicó
- Estados: PENDING, ACCEPTED, REJECTED
- Ver detalles de cada candidatura
- Chat básico (placeholder)

## 4. ProfileScreen - Datos Personales
- Nombre, email, ciudad
- Editar perfil
- Cambiar contraseña
- Cerrar sesión

## 5. EconomicsScreen - Estadísticas (Nueva)
- Ingresos totales
- Días trabajados
- Candidaturas enviadas
- Tasa de aceptación
- Gráficos básicos

## Prioridad de Implementación:
